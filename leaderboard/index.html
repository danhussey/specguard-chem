To submit your model's evaluation:

  1. Run the evaluation: 
    specguard-chem run --suite all --model your_model
  2. Generate submission file: 
    specguard-chem export-leaderboard --results results.jsonl
  3. Open a pull request adding your submission.json to the leaderboard/data/submissions/ directory

Requirements: Complete evaluation on all 54 tasks, reproducible results, model details.
